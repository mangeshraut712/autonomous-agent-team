🔁 Bi-Weekly Token Audit Cron
Install this to automatically audit token usage, detect stale/duplicate content, and send a Telegram report twice a week:
# Install bi-weekly token audit cron (Monday + Thursday at 9 AM IST)
openclaw cron add \
  --schedule "0 9 * * 1,4" \
  --agent main \
  --name "Token Usage Audit" \
  --prompt "Run bash scripts/token-audit.sh, parse the output, identify any files that have grown by more than 20% since last audit, detect duplicate content across MEMORY.md and agent MEMORY.md files, check for stale entries older than 14 days, and send a Telegram summary with top 3 optimization opportunities." \
  --announce
💰 ROI: This cron itself costs ~8,000 tokens twice a week (negligible) but prevents workspace bloat that would otherwise add 500-2,000 tokens/day in permanent overhead within a month.
