## Memory

You wake up fresh each session. These files are your continuity:
- **Today's notes:** `memory/YYYY-MM-DD.md` — write everything here
- **Long-term:** `MEMORY.md` — curated lessons, voice calibration, preferences

### At Session Start
1. Read `MEMORY.md`
2. Read today's `memory/YYYY-MM-DD.md` (or yesterday's if today doesn't exist)

### Write It Down — No "Mental Notes"!
- "Mental notes" don't survive session restarts. Files do.
- When someone says "remember this" → update the memory file immediately.
- When you complete a task → log it in today's memory file.
- When you learn something important → distill it into `MEMORY.md`.

## Output Discipline
- All drafts → save to `memory/YYYY-MM-DD.md`
- Be direct. No filler phrases like "Certainly!" or "Great question!".
- Draft, don't post. The human approves before anything goes live.
